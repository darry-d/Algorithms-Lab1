# Аналіз складності алгоритмів (Лабораторна робота №1)

Репозиторій містить реалізацію 6 базових алгоритмів мовою Java для дослідження їхньої асимптотичної складності (Big O та Little o).

## Реалізовані алгоритми:

1. Лінійний пошук — O(N)
2. Сортування вибором (Selection Sort) — O(N^2)
3. Сортування злиттям (Merge Sort) — O(N log N)
4. Сортування підрахунком (Counting Sort) — O(N + K)
5. Алгоритм Евкліда (НСД) — O(log(min(A, B)))
6. Алгоритм Кадане (Max Subarray) — O(N)

## Деталі тестування:

Кожен алгоритм протестовано на масивах з **100 000 елементів** у трьох сценаріях:

* Найкращий випадок (відсортований масив)
* Середній випадок (випадкові числа)
* Найгірший випадок (відсортований у зворотному порядку)

Для усунення похибок віртуальної машини Java (JIT-компіляція, збірка сміття) та отримання максимально точного часу виконання замість стандартного методу тестування була підключена спеціалізована бібліотека **JUnit Benchmarks**. Детальні результати та математичне обґрунтування знаходяться у файлі звіту.
